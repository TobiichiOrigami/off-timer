# 🕒 下班倒數計時器 (Off-Timer)

一個極簡、現代化的網頁小工具，協助上班族計算預計下班時間，並提供視覺化的倒數計時功能。

![專案預覽圖](./pictures/demo.png)

## ✨ 特色 (Features)

*   **極簡設計**：採用玻璃擬態 (Glassmorphism) 風格，介面清爽現代。
*   **自動計算**：輸入打卡時間，自動推算 9 小時後的下班時間。
*   **動態倒數**：大字體顯示剩餘時間，並隨時間變換鼓勵語句。
*   **分頁標題連動**：瀏覽器分頁標題會同步顯示倒數時間，工作時切換分頁也能隨時掌握進度。
*   **響應式設計**：支援手機與電腦瀏覽。

## 🛠️ 技術棧 (Tech Stack)

本專案保持輕量化，不需複雜建置流程，直接由瀏覽器執行。

*   **HTML5**：語意化標籤結構。
*   **CSS3**：
    *   **Tailwind CSS (CDN)**：快速構建現代化樣式。
    *   **Custom CSS**：實現玻璃擬態與客製化動畫。
*   **JavaScript (Vanilla)**：原生 JS 處理時間邏輯與 DOM 操作。
*   **FontAwesome**：提供精美圖示。

## ⚙️ 功能說明

1. 設定時間：開啟網頁後，輸入您今日的打卡時間（預設為當前時間）。
2. 開始倒數：點擊「開始計算」，系統會自動加上 9 小時工時。
3. 狀態提示：
   - 剩餘 > 4 小時：「新的一天，保持專注 💪」
   - 剩餘 < 4 小時：「午後時光，喝杯咖啡吧 ☕」
   - 剩餘 < 1 小時：「最後衝刺！再撐一下！🔥」
   - 時間到：「下班啦！快回家休息吧！🎉」
